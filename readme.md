# Prosta wersja
Ściągnąć paczkę
```sh
    git clone https://github.com/meridPL/zadanie.git
```

Przejść do katalogu
```sh
    cd .\zadanie\
```

W głowny katalogu po ściągnięciu paczki
```sh
    npm install
```

Nasepnie instalujemy reszte paczek w naszych folderach po przez

```sh
    npm run install-all
```

Następnie odpalamy cały projekt

```sh
    npm run dev
```
